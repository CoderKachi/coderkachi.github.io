K L V L G E N

A 3D Grid-Based Procedural Level Layout Generator Framework for Indie Game Development.

C#
Unity Engine
Game Development
University
2024

Kinetik LVLGen is a level layout generation framework built for the Unity Engine. It offers a straightforward way of configuring and generating level layouts.

This framework includes various components for layout generation, prebuilt steps to produce complete level layouts and a flexible pipeline system that accepts both runtime and developer created generation procedures.
