K T X T

A demonstration of real-time chat within a web application wrapped as a mini game.

JavaScript
mySQL
Node.js
Web Development
University
2023

TXT allows users to interact through a global chat. Users will be required to register an account and login in order to send messages and add other users as friends (however the global chat can still be spectated regardless of login status). Users gain score by being the first to type the word sent by the server.





Kinetik TXT utilised various architectures.

Frontend (Presentation Tier)
The frontend was developed using HTML, CSS and JavaScript. EJS is the templating engine which I used to generate content dynamically and condense code.

Backend (Application Tier)
The backend was developed using Node.js and Express.js. It authenticates users, interacts with the database and deals with requests.

Communication
Socket.io was used to achieve real time bidirectional communication between the server and clients via websockets. It is also compatible with express-session.

Database (Data Tier)
mySQL is used as the relational database to store data (user data, friendships and useful procedures)

Security
Bcrypt is used to hash user passwords and express-validator, express-sanitizer and validator were used for data validation / sanitation.
